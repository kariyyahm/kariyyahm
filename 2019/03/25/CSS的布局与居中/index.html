其他小技巧

块级元素添加 display： inline-block; 后，会发生什么？

  • 块级元素宽度收缩

    块级元素不写宽度的情况下，默认宽100%（占据一行）。

    加入inline-block，宽度自动适合内容。

  • 块级元素下方会产生空隙

    解决方法：添加vertical-align: top;，清除空隙效果。
